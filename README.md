# Aarogya-vaani

Aarogya-vaani architecture inspired from Delhi Sudarshan and an ehanced version of delhi sudarshan to help asha workers(Frontline Health workers)
- Google Sheets for records handling
- Ml_model for incentive calculation
- Community Command center
- Locally train Ml_model for image verfication (Replacing Gemini API key)
- A zero-ui interface for asha wokers as in village there is still no 4G,5G infrastructure
- An app for asha workers (Like Zomato delivery partner app) to track thier incentives
- To protect data, most of the work is done by locally train model
- The AI automatically track medicine stock used by asha workers and order medicine when it is low on stock
